## Community

This repository just contains the documentation hosted on https://docs.folivora.ai Come to https://community.folivora.ai for any feature requests/discussions/bugreports.

## BetterTouchTool Documentation

The BetterTouchTool documentation can be found [here](https://docs.folivora.ai/). Pull requests on the BetterTouchTool documentation (which is hosted on this repo) are always very welcome.

To build the BetterTouchTool Documentation follow these steps:

* cd into the BetterTouchToolDocs directory and run ``npm install``
* run ``npm start`` to start the local development server
* run ``npm run build`` to create a production build
