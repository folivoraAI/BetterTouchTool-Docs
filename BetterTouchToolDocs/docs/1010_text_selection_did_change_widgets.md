# Example Text Selection Widgets To Download

The full preset described in [Text Selection Did Change](1009_text_selection_did_change.md) is always available here:

https://folivora.ai/releases/text-selection-base.bttpresetzip

On this page I'm collecting specific widgets I created or that have been created by users on https://community.folivora.ai. 
In general it's always a good idea to request a specific widge there, it will usually be pretty easy to create one.

## Text Selection Widget Collection

You can download these and drag them into your floating menu to import (they are JSON files that you could also edit in a text editor)
