# Update Menu / Item Properties

You can use the predefined actions "Update Floating Menu Item / Menu Properties" to change the content / appearance of an existing floating menu.

# Updating a Menu Item Icon on Hover
A good example use case is to update the icon of a menu item on hover. To do this you need to configure the "Update Floating Menu ITEM Properties" action to trigger "on hover" like this:
![hover_start](/media/hover_start.png)

To reset the icon after the hover ends, you can use the predefined action "Reset Floating Menu / Item Properties" and assign it to trigger on "hover end"

![hover_start](/media/hover_end.png)
